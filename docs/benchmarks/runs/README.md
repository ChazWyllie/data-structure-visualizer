# Benchmark Runs

This directory contains individual benchmark run logs.

## Naming Convention

Files should be named: `YYYY-MM-benchmark.md`

For multiple runs in a month: `YYYY-MM-benchmark-2.md`

## Creating a New Run Log

1. Copy `../TEMPLATE.md` to this directory
2. Rename to match the date
3. Fill in metrics during/after your benchmark run
4. Commit with the benchmark branch
